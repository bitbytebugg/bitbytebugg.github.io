PDF电子书添加书签的自动化探索

Sep 27, 2024
阅读时长: 3 分钟

背景

有些扫面版的电子书会有这种形式的书签，这看起来不是手动添加的，更像是通过扫描程序自动生成的。这时我就有了一个想法，能否通过程序实现自动添加PDF电子书的书签。

pypdf

目前找到一款软件包pypdf支持添加书签操作。

1

add_outline_item(title, num, parent)

该API中第一个参数为书签文本，第二个参数为书签定位的页码，第三个参数为书签的父节点。

半自动化实现

获取书本目录原始信息

目前还是通过手动方式解决，主要是因为不同网站的目录信息没有一个统一格式。获取途径主要是这几种方法：

  1. 检索电商网站的书籍信息，一般在商品详情页中会有文本格式的目录信息
  2. 检索出版社网站的书籍信息，一般会有书籍详情页会有文本格式的目录信息
  3. 检索百科网站，有些书籍会有目录信息

将原始目录处理为统一格式

统一格式是为了方便后续程序处理，这里是一个统一目录格式的示例：

1
2
3

第1章 {一级书签标题} {页码}
1.1 {二级书签标题} {页码}
1.1.1 {三级书签标题} {页码}
  • “第"和"章"是定位一级标题的关键字
  • 标题和页码之间用空格分隔
  • “.” 是定位二级/三级/N级标题的关键字，如果出现N-1个”.“字符，就认为该书签是N级标签
  • 二级以上书签中的标题文本和前面的级别序号如"1.1"之间必须有空格分隔

'''
https://github.com/py-pdf/pypdf
书签格式示例：
第1章　xv6安装使用 16     # 第x章为一级标签
1.1　运行于QEMU的xv6 16  # 1.1 中dot的个数n指明为n+1级别子书签
1.2　调试观察 19         # 页码右边不能出现另外的数字
'''
import os.path

import pypdf
import sys

in_file = sys.argv[1]
out_dir = r"pdf-bookmark"
# 统一格式的书籍目录信息
bookmark_txt = r"web_book_content.txt"
# 注意实体书的第一页从正文开始算起，电子书的第一页从封面开始算起
offset = int(sys.argv[2])  # pdf内的页码 - 实体书的页码

def main():
    # 将PDF读取到writer中
    in_pdf_rf = open(in_file, "rb")
    reader = pypdf.PdfReader(in_pdf_rf)
    writer = pypdf.PdfWriter()
    for i in range(len(reader.pages)):
        writer.add_page(reader.pages[i])
    # 添加bookmark
    with open(bookmark_txt, "r", encoding="utf-8") as rf:
        lines = rf.readlines()
    # 存储书签父节点的栈
    parent_stack = [None]
    for li in lines:
        li = li.strip()
        if li == "":
            continue
        for i_not_digit in range(len(li) - 1, -1, -1):
            if not li[i_not_digit].isdigit():
                break
        # 解析页码信息
        num = li[i_not_digit+1:].strip()
        # pypdf的页面是从0开始计数的
        num = int(num) + offset - 1
        title = li[:i_not_digit+1].strip()
        # 处理一级书签
        if "第" in title and "章" in title:
            parent = writer.add_outline_item(title, num, parent=parent_stack[0])
            if len(parent_stack) < 2:
                parent_stack.append(parent)
            else:
                parent_stack[1] = parent
        else:
        # 处理二级及以上级别书签
            n_dot = li.split(" ")[0].count(".")
            parent = writer.add_outline_item(title, num, parent=parent_stack[n_dot])
            if len(parent_stack) < n_dot + 2:
                parent_stack.append(parent)
            else:
                parent_stack[n_dot + 1] = parent

    # 输出新的pdf文档
    in_file_name = os.path.basename(in_file)
    out_file_path = os.path.join(out_dir, in_file_name)
    out_pdf_wf = open(out_file_path, "wb")
    writer.write(out_pdf_wf)
    # 关闭所有打开的资源
    writer.close()
    out_pdf_wf.close()
    in_pdf_rf.close()
    print("Success out to {}".format(out_file_path))

if __name__ == '__main__':
    main()
